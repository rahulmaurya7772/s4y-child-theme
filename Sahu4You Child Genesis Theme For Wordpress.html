Install S4Y Child Genesis Theme

  • Upload the S4Y Child theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
  • Go to your WordPress dashboard and select Appearance.
  • Activate the S4Y Child theme.
  • Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.
